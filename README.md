# 程序结构
- vue.js组件化开发
- webpack构建、打包
- iViewUI框架
- axios数据请求
- [mavonEditor](https://github.com/hinesboy/mavonEditor)组件实现markdown功能
- pug+scss+flex开发页面布局

npm run build
